<Callout>
    **Note:** Follows the same installation steps as <a href="https://ui.shadcn.com" target="_blank">shadcn/ui</a>
</Callout>

<Steps>
    <Step>
        ### Create project
    </Step>
    Run the `init` (with default options) command to create a new Next.js project or set up an existing one:

```npm
npx shadcn@canary init
```
    <Step>
        ###  Add `@paceui-ui` to your `components.json`
    </Step>

```json
{
    ...
    "registries": {
        /* [!code ++:1] */
        "@paceui-ui": "https://ui.paceui.com/r/{name}.json"
    }
}
```

    <Step>
        ### That's it
    </Step>
    Youâ€™re ready to start adding components to your project

```npm
npx shadcn@canary add @paceui-ui/text-reveal
```

    The command above will add the `TextReveal` component to your project. You can then import it using the following syntax:

```tsx
{/* [!code ++:1] */}
import { TextReveal } from "@/components/gsap/text-reveal"

export default function Home() {
    return (
        <TextReveal>PaceUI</TextReveal>
    )
}
```

</Steps>
