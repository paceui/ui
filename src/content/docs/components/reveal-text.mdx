## Props

### RevealText

| Prop          | Type                    | Default |
|---------------|-------------------------|---------|
| type          | `chars` `words` `lines` | chars   |
| gsapVars      | gsap.TweenVars          | -       |
| splitTextVars | SplitText.Vars          | -       |
