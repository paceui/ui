## Props

### DrawLineText

| Prop          | Type    | Default |
|---------------|---------|---------|
| text          | string  | -       |
| afterFill     | boolean | true    |
| oneByOne      | boolean | true    |
| fontSize      | number  | 40      |
| wordSpacing   | number  | 10      |
| strokeWidth   | number  | 1       |
| letterSpacing | number  | 0       |
| color         | string  | -       |
